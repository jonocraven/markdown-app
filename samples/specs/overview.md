# Specs overview

This is the specs-folder overview — one half of a deliberately ambiguous
filename pair. Its stem, `overview`, is shared with
[the notes-folder overview](../notes/overview.md); a relative link like this
one is unambiguous (it names the full path), but a wikilink to `[[overview]]`
is not — see [the vault index](../index.md) for that test case.

See also the [[api spec]] and [[design|the design note]].
