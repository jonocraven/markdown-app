# API spec

The reference API for the local-first core. Linked from [the vault index](../index.md) and from [the specs overview](./overview.md).

## Endpoints

- `GET /files` — list every markdown file under the root.
- `POST /files` — create a file (used by the broken-wikilink create-on-click flow).

Jump straight there with an in-page anchor: [Endpoints](#endpoints).

## Authentication

Nothing fancy — local-first, no auth in v1.
